样式定义如何显示HTML元素

样式通常存储在样式表中

####语法
两个主要部分构成：

>选择器

>一条或多条声明

选择器通常是需要改变样式的HTML元素。

####选择器的分组

你可以对选择器进行分组，这样，被分组的选择器就可以分享相同的声明。用逗号将需要分组的选择器分开。在下面的例子中，我们对所有的标题元素进行了分组。所有的标题元素都是绿色的。
````html
h1,h2,h3,h4,h5,h6 {
  color: green;
}
~~~~

子元素从父元素继承属性


####派生选择器
通过依据元素在其位置的上下文关系来定义样式，可以使用标记更加简洁。

后代选择器、子元素选择器、相邻兄弟选择器。
