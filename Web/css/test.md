####CSS背景
- 1、设置背景颜色：background-color:red;
- 2、图片设为背景：background-image:url('');选择浅色图片作为背景图片（影响文字的查看）
- 3、在某个方向上重复背景图片：background-repeat:repeat-x;
- 4、背景图片的定位：background-position:right top;
- 5、固定的背景图片：background-attachment:fixed;
- 6、可以将上述一起声明：background:#ffffff url('img_tree.png') no-repeat right top;

####CSS文本
- 1、设置不同元素的文本颜色：color:red;color:#00ff00;color:rgb(0,0,255);
- 2、文本对齐：text-align:center;
- 3、移除文本的下划线：text-decoration:none;
- 4、装饰文字；text-decoration:overline;text-decoration:line-through;--underline;
- 5、控制文本中的字母（大小写）：text-transform:uppercase;--lowercase;--capitalize;
- 6、缩进文本：text-indent:50px;
- 7、指定字符间的空间：letter-spacing:-2px;
- 8、指定行之间的距离：line-height:70%;
- 9、设置元素的文本方向：direction:rt1;
- 10、增加单词之间的空格：word-spacing:30px;
- 11、在一个元素内禁用文字换行：white-space:nowrap;
- 12、内部文字图像的垂直对齐：vertical-align:text-top;--text-bottom;
